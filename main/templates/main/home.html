Welcome to Adamas Audio



This website is designed to allow fast and cheap production of Audiobooks. A full elaboration is given on the medium article Adamas Audio Medium Article .

The Conventional Audio will be divided into 2 choices: Default and Custom controls; in the Default mode the default audio settings will be used. In Custom controls, the audio controls will be asked for as inputs for the user to provide and a custom audio file/ audiobook will be generated as per user specification.

The Voice Synthesis will require an audio file of any users voice one would like to replicate with a max size of being 20 MB's due to current technological limitations on the site. This in combination of the text provided should generate a synthetic voiceover. Current limit will be 6 pages per click, feel free to use this feature as many times as you like. Results may vary according to audio quality provided to be trained.

The PDF Compression tool will compress PDF's to the maximum extent possible without losing quality.

The Audio Merger tool will merge audio files into 1, provided the uploaded files are mp3 files. Further extensions maybe offered in the future depending on site performance.

Future features that maybe added depending on how much user interactivity the website recieves:



i) Audio cleaning tools: in order to clean, segregate and label audio with multiple voices inside it, so each one can be used as base audio trainer to produce synthetic audiobooks per user specification.

ii) PDF splitting and PDF merging tools, so users can rapidly feed text data into the Voice synthesis feature, and can help synthetic audiobook or audioscript production.

iii) Increase processing rates and audio training data size for Voice synthesis; this is expensive as RAM is expensive and can only be implemented once the website has sufficient users.

If you would like to add, additional features please go ahead and use the website url to tweet about it, once a contact form is streamlined; I'll be more than happy to hear and try to implement user input. This will be the current designated twitter account: Adamas Audio . Feel free to tweet feedback at the account, I'll strive to go through all the feedback. Feel free to share across the internet, the more feedback the better the product will be in the future.
